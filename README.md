# CLI TCP Chat
*(Made and published without major reason)*
## Description
Small and simple CLI chat built with Go.
## Features
- Server and client
- Username choosing
- Easy-to-add chat commands
## Get started
1. Clone repo:
```bash
git clone https://github.com/Wrtgvr2/tcp-chat.git
cd tcp-chat
```
2. Run server:
```bash
go run ./server/
```
3. Run client(-s):
```bash
go run ./client/
```
